# Assets mobile

Coloque aqui os arquivos fonte para gerar icones e splash do Capacitor:

```text
icon-only.png
icon-foreground.png
icon-background.png
splash.png
splash-dark.png
```

Depois rode:

```bash
npm --workspace apps/web run mobile:assets
```

Sugestao visual:

- Fundo: `#07090d`
- Verde principal: `#21d2ad`
- Laranja de apoio: `#f78c6b`
- Icones: 1024x1024 PNG
- Splash: 2732x2732 PNG
